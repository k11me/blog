jekyll 学习指南

Posted on 2016-07-12 | In ubuntu |

1.jekyll

jekyll官方地址

jekyll中文

1.1安装jekyll必要组件

下文是在ubuntu上安装与设置

1
$ sudo apt-get install ruby ruby-dev make gcc nodejs

1.2安装jekyll

1
$ sudo gem install jekyll

之后查看jekyll版本

1
$ jekyll -v

1.3jekyll使用

首先要设置_config.yml文件

随后build为site。

1
$ jekyll build

build后网站文件在_site

使用jekyll server运行测试。

1
$ jekyll server

浏览器输入localhost:4000可以查看网站运行情况。

_post文件下位文章列表，可以将文章放入后build。

注意：文件的头部要使用下方格式

1
2
3
4
---
layout: post
title: Blogging Like a Hacker
---
